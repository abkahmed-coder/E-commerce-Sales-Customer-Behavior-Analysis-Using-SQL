# Website Traffic & Sales Performance Dashboard

## Project Overview

This project analyzes website traffic, customer behavior, marketing performance, and sales outcomes for an e-commerce business using Power BI and Excel. The dashboard combines website analytics and transaction data to provide insights into customer acquisition, conversion performance, product sales, and refund activity.

The analysis was performed on a dataset containing over **1.5 million records** distributed across **6 worksheets**, requiring data cleaning, modeling, and relationship building before visualization.

---

## Business Objectives

The dashboard was developed to answer the following business questions:

- Which traffic sources generate the most website sessions?
- Which channels achieve the highest conversion rates?
- How effective are marketing campaigns in driving sales?
- What percentage of visitors return to the website?
- Which products generate the highest number of orders?
- Which products have the highest refund requests?
- How have website sessions, orders, and revenue changed over time?

## Data Preparation

The following steps were completed before analysis:

- Imported and combined data from multiple worksheets.
- Removed duplicate records.
- Handled missing values.
- Standardized data formats.
- Created calculated columns and measures.
- Built relationships between transactional and behavioral datasets.
- Optimized the data model for reporting performance.

---

## Data Model

The Power BI data model was designed using a relational structure connecting website activity, sales transactions, products, and refunds.

### Key Relationships

- Website Sessions → Orders
- Orders → Order Items
- Orders → Refunds
- Products → Order Items
- Website Sessions → Website Pageviews

This structure enables end-to-end analysis from customer acquisition through purchase and refund activity.

## Key Insights

### Traffic Performance
- Organic search generated the highest number of website sessions.
- Direct traffic delivered the highest conversion rate.

### Customer Behavior
- Returning visitors represented 16.61% of total website traffic.
- Desktop users accounted for approximately 86% of total orders.

### Product Performance
- A small group of products generated the majority of orders.
- Certain products showed significantly higher refund activity compared to others.

### Revenue Growth
- Revenue increased consistently over the analyzed period.
- Website sessions and total orders followed a similar upward trend, indicating healthy business growth.
